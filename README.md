# HabitTracker

React app that prompts every 30 minutes to log activities with dark mode UI.

## Quick Start

```bash
bun install
bun run dev
```

Login with `admin@tracker.com` / `Admin@123`

## Deploy to Vercel

```bash
git init && git add . && git commit -m "Initial commit"
git push
```

Import repo on Vercel, add env vars, deploy.
